# JavaScript Projects

These are a collection of small projects I built using JavaScript to strengthen my understanding of front-end development and improve my coding skills.

## Projects

- Pizza Menu  
- Tic-Tac-Toe Game  
- Calculator  

## Pizza Menu

This project is an interactive pizza ordering application that allows users to select different pizza sizes and toppings. The application dynamically calculates the total price based on user selections.

It demonstrates my understanding of:
- Form handling  
- User input processing  
- Dynamic price calculation using JavaScript  

## Tic-Tac-Toe Game

This is a web-based version of the classic tic-tac-toe game. The game randomly determines which player goes first and allows users to choose their symbols. Players can take turns, and the game detects the winner with visual feedback.

It demonstrates my understanding of:
- Game logic implementation  
- DOM manipulation   

## Calculator

A simple and responsive calculator that performs basic arithmetic operations such as addition, subtraction, multiplication, and division. The interface updates dynamically based on user input.

It demonstrates my understanding of:
- JavaScript functions  
- Input handling  
- Real-time display updates
